Quick Decision Rules

‚úÖ USE Multilevel Modeling IF:

  • ICC ‚â• .05 (any meaningful clustering)
  • Data are hierarchically structured
  • You want to test cross-level interactions
  • Relationships vary across groups
  • You have longitudinal/repeated measures data
  • Your research questions involve multiple levels

‚ùå DON'T USE Multilevel Modeling IF:

  • No nested structure (truly independent observations)
  • ICC < .05 AND no theoretical reason for MLM
  • You have only one Level-2 unit (can't estimate between-group variance)
  • Sample sizes too small (see below)

Minimum Sample Size Guidelines

Level 2 (Groups):

  • Minimum: 30 groups
  • Preferred: 50+ groups
  • For cross-level interactions: 100+ groups

Level 1 (Individuals per group):

  • Minimum average: 10 per group
  • Can vary across groups
  • Total Level-1 n depends on design

Power Considerations:

  • More Level-2 units > More Level-1 units per group
  • 50 groups with 10 each (n=500) > 10 groups with 50 each (n=500)

RESOURCE 7: MODEL SPECIFICATION CHECKLIST

Pre-Analysis Checklist

Data Preparation:

  • [ ] Data file properly structured (long format for MLM)
  • [ ] Level-1 and Level-2 variables clearly identified
  • [ ] Missing data pattern examined
  • [ ] Outliers identified and decision made
  • [ ] Variables centered as planned

Descriptive Statistics:

  • [ ] Means, SDs, ranges for all variables at each level
  • [ ] Correlation matrix computed
  • [ ] Multicollinearity checked (VIF < 10)
  • [ ] Sample sizes at each level documented

Model Building Sequence Checklist

Step 1: Null Model (Unconditional)

  • [ ] Outcome variable only, no predictors
  • [ ] ICC calculated
  • [ ] Variance components reported (œÑ‚ÇÄ‚ÇÄ, œÉ¬≤)
  • [ ] Decision: Is MLM needed?

Interpretation Questions:

  • What % of variance is between groups?
  • Is this substantial enough to warrant MLM?

Step 2: Random Intercept Model (Level-1 Predictors)

  • [ ] Level-1 predictors added
  • [ ] Intercepts still random
  • [ ] Slopes fixed (constrained equal across groups)
  • [ ] Model fit compared to null model

Interpretation Questions:

  • Which Level-1 predictors are significant?
  • How much Level-1 variance is explained?
  • Do Level-2 intercepts still vary significantly?

Step 3: Random Slopes Model

  • [ ] Key slopes allowed to vary
  • [ ] Variance in slopes tested
  • [ ] Model fit compared to random intercept model
  • [ ] Covariance between random effects examined

Interpretation Questions:

  • Do slopes vary significantly across groups?
  • Which relationships are consistent vs. variable?
  • Is there covariance between intercepts and slopes?

Step 4: Full Model (Level-2 Predictors & Cross-Level)

  • [ ] Level-2 predictors added to explain intercept variance
  • [ ] Level-2 predictors added to explain slope variance (if slopes random)
  • [ ] Cross-level interactions tested
  • [ ] Final model compared to previous models

Interpretation Questions:

  • What explains between-group differences in intercepts?
  • What explains between-group differences in slopes?
  • Are there significant cross-level interactions?

Assumption Checking Checklist

Before Running Models:

  • [ ] Linearity of Level-1 relationships
  • [ ] No extreme multicollinearity
  • [ ] Adequate sample sizes
  • [ ] No extreme outliers (or robust methods used)

After Running Models:

  • [ ] Normality of Level-1 residuals (histogram, Q-Q plot)
  • [ ] Normality of Level-2 residuals (random effects)
  • [ ] Homoscedasticity of residuals (plot)
  • [ ] Independence of residuals within groups

If Assumptions Violated:

  • [ ] Transformations considered
  • [ ] Robust standard errors used
  • [ ] Limitations acknowledged
  • [ ] Alternative models explored

Common Mistakes Checklist

Avoid These Errors:

  • [ ] ‚ùå Ignoring significant clustering (ICC > .05)
  • [ ] ‚ùå Treating Level-2 variables as Level-1
  • [ ] ‚ùå Not centering predictors appropriately
  • [ ] ‚ùå Failing to justify centering decisions
  • [ ] ‚ùå Using level for predictors instead of where they vary
  • [ ] ‚ùå Reporting only fixed effects (forgetting random effects)
  • [ ] ‚ùå Not testing if slopes should be random
  • [ ] ‚ùå Adding too many random effects with small samples
  • [ ] ‚ùå Comparing models with different fixed effects using deviance
  • [ ] ‚ùå Not checking assumptions

MULTILEVEL MODELING ESSENTIALS

When to Use:

  • Nested/hierarchical data (ICC > .05)
  • Cross-level interactions
  • Varying slopes across groups
  • Longitudinal data

Key Concepts:

  • ICC = œÑ‚ÇÄ‚ÇÄ / (œÑ‚ÇÄ‚ÇÄ + œÉ¬≤) = % variance between groups
  • Fixed effects (Œ≥) = Average effects across all groups
  • Random effects (œÑ, u) = Variance in effects across groups
  • Level 1 = Within-group variation
  • Level 2 = Between-group variation

Centering Choices:

  • Grand-mean = Subtract overall mean (for between-group effects)
  • Group-mean = Subtract group mean (for within-group effects)
  • Uncentered = Raw scores (intercept = Y when X=0)

Model Building:

  1. Null model ‚Üí Calculate ICC
  2. Add Level-1 predictors ‚Üí Random intercepts
  3. Test random slopes ‚Üí Allow slopes to vary
  4. Add Level-2 predictors ‚Üí Explain variance
  5. Test cross-level interactions ‚Üí Moderation

Software:

  • HLM: User-friendly GUI
  • R (lme4): Flexible, free
  • Stata (mixed): Powerful
  • Mplus: SEM integration

Report:

  • Fixed effects (Œ≥) with SE
  • Random effects (œÑ) with SE
  • Model fit (AIC, BIC)
  • ICC and variance explained
  • Sample sizes

MIXED METHODS ESSENTIALS

Design Types:

  • QUAL ‚Üí quan: Exploratory Sequential (explore then measure)
  • QUAN ‚Üí qual: Explanatory Sequential (measure then explain)
  • QUAN + qual: Convergent (compare/triangulate)
  • QUAN(qual): Embedded (one supports other)

Integration Methods:

  • Merging: Joint displays, side-by-side comparison
  • Connecting: Phase 1 informs Phase 2 sampling/questions
  • Embedding: One nested within the other

Key Integration Points:

  • Data collection
  • Analysis
  • Interpretation
  • Throughout (embedded)

Quality Criteria:

  • Design appropriateness
  • Strand rigor (QUAN + QUAL)
  • Integration quality
  • Meta-inferences

Joint Display Elements:

  • Quantitative findings
  • Qualitative findings
  • Meta-inference (synthesis)
  • Convergence/divergence noted

RESOURCE 9: LITERATURE REVIEW SYNTHESIS GUIDE

From Article Summaries to Synthesis (Chapter 2)

The Problem:

Many doctoral students write Chapter 2 as a series of article summaries rather than a true synthesis. This fails to demonstrate scholarly understanding and makes a weak foundation for the dissertation.

Article Summary (‚ùå Don't Do This):

> Smith (2020) studied teacher retention in urban schools. He found that mentoring improved retention rates. Jones (2019) also studied retention. She found salary was important. Williams (2021) looked at working conditions.

Synthesis (‚úÖ Do This Instead):

> Teacher retention in high-poverty schools is influenced by multiple interconnected factors. While compensation serves as a necessary threshold (Jones, 2019; Williams, 2021), research consistently identifies supportive leadership and mentoring as more decisive retention factors (Smith, 2020; Davis, 2019). Notably, the interaction between these factors varies by career stage: early-career teachers prioritize mentoring and professional growth (Smith, 2020), while mid-career teachers emphasize autonomy and leadership opportunities (Williams, 2021). This suggests retention interventions should be differentiated by teacher experience level.

Synthesis Strategies

Strategy 1: Thematic Organization

Instead of: Author by author

Do: Theme by theme with multiple authors per theme

Example Structure:

  • Theme 1: Individual Factors (Personal characteristics, motivation, preparation)
  • Theme 2: Organizational Factors (Leadership, culture, resources)
  • Theme 3: External Factors (Policy, labor market, community)

Each theme synthesizes across studies, noting patterns, contradictions, and gaps.

Strategy 2: Identify Patterns

  • Convergent findings: What do most studies agree on?
  • Divergent findings: Where do studies contradict?
  • Gaps: What hasn't been studied?
  • Methodological limitations: What approaches need strengthening?

Synthesis Example:

> Research converges on the importance of principal support for teacher retention (15 studies reviewed), yet findings diverge regarding the specific leadership behaviors that matter most. Quantitative studies emphasize instructional leadership (Smith, 2020; Davis, 2021), while qualitative research highlights emotional support and trust-building (Chen, 2019; Williams, 2021). This divergence may reflect methodological differences: survey instruments capture observable behaviors while interviews reveal relational dimensions. No studies have used mixed methods to examine both simultaneously‚Äîa gap this dissertation addresses.

Strategy 3: Build Toward Your Study

Each section of Chapter 2 should build the argument for WHY your study is needed.

Structure:

  1. What we know (synthesize existing research)
  2. What we don't know (identify gaps)
  3. Why it matters (significance)
  4. How your study addresses the gap

Strategy 4: Use Transitional Synthesis

Between major sections, write synthesis paragraphs that:

  • Summarize what was just covered
  • Identify tensions or unresolved questions
  • Bridge to the next section

Example:

> The preceding review establishes that teacher retention is multiply determined by individual, organizational, and policy factors. However, most research treats these factors independently, failing to examine their interactions. Moreover, quantitative studies dominate the literature, leaving the mechanisms by which factors influence retention poorly understood. The following section examines the limited mixed-methods research that has begun to address these gaps.

Synthesis Writing Techniques

Technique 1: Cluster Citations

Instead of: (Smith, 2020). (Jones, 2019). (Davis, 2021).

Do: (Davis, 2021; Jones, 2019; Smith, 2020)

Group related findings together with semicolon-separated citations.

Technique 2: Comparison Language

Use words that show relationships:

  • Similarly, likewise, in agreement
  • However, in contrast, conversely
  • Moreover, furthermore, additionally
  • Despite, although, whereas

Technique 3: Critical Analysis

Don't just report‚Äîanalyze:

  • Methodological critique: "While Smith's (2020) survey provides breadth, the cross-sectional design cannot establish causality."
  • Theoretical critique: "Jones (2019) applies social capital theory, yet fails to account for power differentials in school networks."
  • Gap identification: "No research has examined how retention factors vary across school contexts."

Technique 4: Your Voice Leads

You are the author; sources support YOUR argument:

Instead of: "Smith says...then Jones says...then Davis says..."

Do: "Retention is complex [YOUR claim], as evidenced by [Smith, Jones, Davis]."

Chapter 2 Self-Assessment

Ask yourself:

  • [ ] Can I see the structure without looking at headings?
  • [ ] Does each paragraph have a clear purpose?
  • [ ] Do I synthesize across studies (not summarize one at a time)?
  • [ ] Do I identify patterns, contradictions, and gaps?
  • [ ] Do I critique methodologies and theories?
  • [ ] Is my voice leading the narrative?
  • [ ] Does the chapter build logically toward my study?
  • [ ] Is the gap my study addresses crystal clear?

If you answer NO to several questions, revise for synthesis.

RESOURCE 10: SAMPLE JOINT DISPLAYS WITH DATA

Example 1: Teacher Professional Development Impact

Research Question: How does PD affect classroom practice, and what explains variation in implementation?

| Teacher | Quantitative: Practice Change Score (Pre-Post) | Qualitative: Implementation Description | Integrated Finding |

|---------|----------------------------------------------|----------------------------------------|-------------------|

| Teacher A | +25 points (Large gain) | "Immediately tried new strategies. Collaborated weekly with peers. Felt supported by principal." | High implementation facilitated by collaboration and admin support |

| Teacher B | +18 points (Moderate gain) | "Used some strategies. Would like more time. Felt rushed in curriculum." | Moderate implementation limited by time constraints |

| Teacher C | +3 points (Minimal gain) | "Interesting ideas but don't fit my students. No follow-up support provided." | Low implementation due to perceived mismatch and lack of support |

Meta-Inference: PD impact varies substantially (3-25 points). Qualitative data reveals implementation success factors: peer collaboration, administrative support, and perceived fit with student needs. Time constraints and lack of follow-up emerge as key barriers.

Example 2: Principal Leadership and School Climate

Research Question: What leadership styles relate to positive climate, and how do principals enact these styles?

| Leadership Style | Climate Score (n=50 schools) | Principal Interview Themes | Synthesis |

|-----------------|----------------------------|---------------------------|-----------|

| Transformational | M = 4.2/5 (Highest) | "Building shared vision," "Empowering teachers," "Inspiring change" | Strong positive relationship; enacted through vision-building and empowerment |

| Instructional | M = 3.9/5 (High) | "Focus on teaching," "Providing feedback," "Data-driven decisions" | Positive relationship; enacted through focus on instructional improvement |

| Managerial | M = 3.1/5 (Moderate) | "Maintaining order," "Following procedures," "Efficient operations" | Weakest relationship; climate benefits from relational not just operational leadership |

Meta-Inference: Transformational leadership shows strongest association with positive climate (quantitative), enacted through vision and empowerment (qualitative). Purely managerial approaches show weakest association, suggesting relational dimensions matter more than operational efficiency for climate.
